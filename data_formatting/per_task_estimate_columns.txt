================================================================================
current_per_task_estimates.xlsx  (one row per (occupation, task))
================================================================================

Task ID
  O*NET task identifier for this row (integer, from Task Ratings).

Task Name
  Wording of the task statement from Task Ratings.

Occupation ID
  O*NET–SOC code for the occupation (from Task Ratings).

Occupation Name
  O*NET occupation title from Task Ratings.

Hosseini et al. exposure estimate
  Binary exposure (0/1) generated by the prompt used by Hosseini et al., or missing if absent.

Alex interview exposure estimate
  Binary exposure (0/1) generated by the method developed by Alexander Wan, or missing if absent.

Expertise classification (1-5)
  Classification of task expertise generated by the prompt used by Hosseini et al., or missing if absent.

Time spent per day estimate
  Computed time spent per day in hours, or missing if absent.

Mean Importance
  Mean O*NET importance rating (IM scale) from Task Ratings.

Mean Importance Renormalized To Be Time Shares
  Renormalized mean importance value so that the mean importance sums to 7 hours across all tasks in an occupation.


================================================================================
current_per_occupation_estimates.xlsx  (one row per occupation)
================================================================================

Occupation ID
  O*NET–SOC code (same universe as Task Ratings rows used to build the file).

Occupation Name
  O*NET occupation title from Task Ratings.

Wage
  National mean annual wage from the bundled OEWS extract, matched on truncated SOC (OCC_CODE vs dotted SOC before the dot), or missing.

Employment
  National total employment from that same OEWS extract, or missing.

Hosseini et al., Occupation Exposure Share
  The weighted fraction of tasks marked as exposed by the Hosseini et al., exposure measure, where the weight for each task is 1 if the task is marked as a Core task in O*NET and 0.5 otherwise.

Hosseini et al., Occupation Exposure Time Share
  The weighted fraction of tasks marked as exposed by the Hosseini et al., exposure measure, where the weight for each task is the computed time per day.

Hosseini et al., Occupation Exposure Importance Weighted
  The weighted fraction of tasks marked as exposed by the Hosseini et al., exposure measure, where the weight for each task is the O*NET mean importance.

Alex, Occupation Exposure Share
  The weighted fraction of tasks marked as exposed by Alexander Wan's exposure measure, where the weight for each task is 1 if the task is marked as a Core task in O*NET and 0.5 otherwise.

Alex, Occupation Exposure Time Share
  The weighted fraction of tasks marked as exposed by Alexander Wan's exposure measure, where the weight for each task is the computed time per day.

Alex, Occupation Exposure Importance Weighted
  The weighted fraction of tasks marked as exposed by Alexander Wan's exposure measure, where the weight for each task is the O*NET mean importance.
